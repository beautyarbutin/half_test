# AGENTS.md

## Purpose

This file is the lightweight entry point for agents working in this repository.

## Read first

Before doing substantive work, read:

- `docs/agent/shared_guidance.md`

If present, also read:

- `LOCAL.md`

That document contains the shared repository guidance for:

- project identity
- context-reading strategy
- GitHub SSH rules
- design direction
- collaboration rules
- working style
- shared UI expectations
- deploy guidance references

`LOCAL.md` is for machine-local or user-specific notes. It is intentionally
ignored by git and must not be required for repository correctness.

## Additional note

If deployment is part of the task, also follow:

- `docs/deploy/safe_deploy.md`
